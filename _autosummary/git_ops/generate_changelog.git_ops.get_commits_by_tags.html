generate_changelog.git_ops.get_commits_by_tags(repository, tag_filter_pattern, starting_tag=None)[source]#

Group commits by the tags they belong to.

Parameters:

  • repository (Repo) – The git repository object

  • tag_filter_pattern (str) – A regular expression pattern that matches valid tags as versions

  • starting_tag (Optional[str]) – Only include tags after this one

Return type:

List[GitTag]

Returns:

A list of dictionaries with tag information with most recent first
